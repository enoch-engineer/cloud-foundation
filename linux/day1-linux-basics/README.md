## Commands Practiced

`wsl --install -d Ubuntu`  
Installed Ubuntu on WSL2

`sudo apt update && sudo apt upgrade`  
Updated and upgraded system packages

`ls`  
Listed files and directories

`pwd`  
Displayed the current working directory

`cd`  
Navigated between directories

`mkdir`  
Created a new directory

`touch`  
Created a file

`nano`  
Edited a file in the terminal

`cat`  
Viewed file contents
